Manual restore

To restore from the command line:

/opt/fsbackup/utils/fs-restore.sh restore --class <class> --id <target> --latest --to /restore/path
/opt/fsbackup/utils/fs-restore.sh restore --class <class> --id <target> --snapshot weekly-2026-W12 --to /restore/path

For S3 archives:

aws s3 cp s3://fsbackup-snapshots-SUFFIX/<tier>/<class>/<target>/<archive>.tar.zst.age . \
  && age -d -i /etc/fsbackup/age.key <archive>.tar.zst.age | zstd -d | tar -xf - -C /restore/path/
